# 幻兽帕鲁钓鱼助手

基于视觉识别的幻兽帕鲁自动钓鱼工具。

QQ群: 162714267

## 使用方法

1. 游戏设置为**窗口模式**
2. **双击运行** `帕鲁钓鱼助手.exe`，主窗口启动
3. 抛竿后按 **F9** 打开预览效果，确认能看到游戏画面（有画面说明捕获正常）
4. 找到钓鱼点，**人工抛竿**，鱼上钩后**点一下鼠标左键**进入钓鱼小游戏
5. 按 **F10** 开启自动钓鱼
6. 按 **F7** 打开小窗查看浮标追踪状态

## 热键

| 按键 | 功能 |
|------|------|
| F7 | 开关小窗查看 |
| F8 | 截图 |
| F9 | 开关预览效果 |
| F10 | 开关自动钓鱼 |
| F12 | 开关参数调整 |

## 调参

按 F12 打开参数面板：
- **前探时间** — 浮标追踪灵敏度，追不上加大，冲过头减小
- **触发阈值** — 控制死区，太敏感抖动则加大
- **自动采样颜色** — 先按 F8 截图，点此按钮，框选绿框和浮标自动更新 HSV
